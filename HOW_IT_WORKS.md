# How It Works

This repository contains everything needed to set up automated Hermes Agent backups.

## Quick Deploy

Share this repository URL with any AI agent (Hermes, OpenCode, Claude Code, etc.) and it can set up the entire backup system automatically:

```
https://github.com/MrElixir67/how-to-backup-hermes-on-your-vps
```

The agent will:

1. Read the README for instructions
2. Download the backup and restore scripts
3. Guide you through SSH key setup
4. Configure the backup target
5. Run a test backup
6. Schedule automatic daily backups

No manual scripting required.
